# Desc

Create a function that takes a Roman numeral as its argument and returns its value as a numeric decimal integer.
You don't need to validate the form of the Roman numeral.

Help:

Symbol | Value
:---:  | :---:
I      |   1
V      |   5
X      |   10
L      |   50
C      |   100
D      |   500
M      |   1,000
